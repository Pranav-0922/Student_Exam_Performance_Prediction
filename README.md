# Student Exam Performance Prediction

## Overview
This project aims to predict student performance in exams using various machine learning techniques.

## Features
- Predicts exam scores
- Analyzes student data
- Visualizes trends

## Setup
### Prerequisites
- Python 3.7+
- Packages:
  - numpy
  - pandas
  - scikit-learn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Pranav-0922/Student_Exam_Performance_Prediction.git
   ```
2. Install required packages:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
To run the prediction model, use:
```bash
python predict.py
```
